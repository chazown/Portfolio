기업 내외부망 보안 시스템 구축 프로젝트 개요

1. 기업 환경 모델링

대상: 가상의 대형 식품기업 모델링

구성: 본사(DNS 할당), 지사(DHCP, Web 운영), 협력사(별개 DNS)

접근: 본사 DMZ 통해 외부 고객 접근 가능

구현: GNS3를 통한 네트워크 가상화 구현

↓
기반 환경 구축
2. 보안 아키텍처 설계

네트워크 분리: 내부망/외부망 분리

방화벽: 각 구간 경계에 pfSense 방화벽 배치

DMZ: 웹 서버, 메일 서버 위치

내부 위협 탐지: 내부망 초입에 IDS 배치

접근 제어: 외부 접근 ACL 정책(포트/서비스 기준) 제어

데이터 무결성: 사용자 인증 정책 설정

↓
보안 체계 확립
3. 취약점 탐지 및 대응 전략 수립

탐지 도구: Metasploit, nmap 활용 모의침투 테스트

탐지 취약점: FTP 익명 접근 허용, 잘못 구성된 SNMP 접근

대응 전략: 포트 차단, 버전 업데이트, 접근 IP 제한 ACL 설정, IDS 정책 강화 등 실행
